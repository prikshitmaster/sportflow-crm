# SportFlow CRM — Claude Token-Saving Build Prompt (Mobile Responsive)

Build a **production-ready mobile-first SaaS CRM** for Sports Academies in India.

## Goal

Help academy owners manage:

* students
* attendance
* fees
* trials
* staff
* announcements
* reports

## Stack

Use modern fast stack:

* Next.js
* Tailwind CSS
* Supabase
* Responsive UI
* Clean reusable components

## Design Rules

* Mobile first
* Fully responsive (320px to desktop)
* Premium SaaS look
* Fast loading
* Simple for non-technical users
* Large tap targets
* Sticky bottom nav on mobile
* Sidebar on desktop
* White / slate / blue theme
